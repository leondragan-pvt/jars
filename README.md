# jars

Just Another React Starter [Kit]

## Features

- Website background pattern from http://www.subtlepatterns.com
- [babel](babel/babel) with es2015 and react presets
- [webpack](webpack/webpack) with dev server


### Running 
- Clone or download repo
`npm run build`
`npm run dev`
