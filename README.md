# jars

Just Another React Starter [Kit]

## Features

- [babel](https://github.com/babel/babel) with es2015 and react presets
- [webpack](https://github.com/webpack/webpack) with dev server


### To run
- Clone or download repo
- `npm run build`
- `npm run dev`
